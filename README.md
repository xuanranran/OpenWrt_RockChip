### 固件默认配置
- 用户名：`root` 密码：`password` 管理IP：`192.168.11.1`
- 下载地址：https://github.com/xuanranran/OpenWrt_RockChip/releases 对应 Tag 标签内下载固件
- 刷机方法请参考dn2刷机 https://github.com/xuanranran/OpenWrt_RockChip/blob/master/data/emmc.md
